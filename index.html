CSS Information

Enjoy learning an overview of some exciting CSS specifications!

Three interesting CSS techniques to be aware of!

CSS Nesting

This enables you to make your CSS mirror your HTML Nesting. For example, having a paragraph with an anchor tag nested inside!

CSS Variables

CSS now supports Custom Properties, aka Variables. This enables you to store font-families in a variable for reuse when inheritance doesn't work. <a></a> 👀 !

Container Queries

This new technique enables you to determine how your card element should react based on it's width and not the Viewport!
